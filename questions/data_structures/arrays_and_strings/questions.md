### ctci-1_1-isUnique

Implement an algorithm to determine if a string has all unique characters.

Part 2: What if you cannot use additional data structures

**Examples**:
```
Input: "abcd"
Output: True

Input: "aabcd"
Output: False

Input: "Aabcd"
Output: True

Input: ""
Output: True
```
